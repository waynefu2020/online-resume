    NPM链接 源码链接

    【项目实施】针对项目中翻译功能和命令行功能进行独立封装。命令行功能使用了 commander.js 设置传入参数。翻译功能使用了 md5 进行 API 鉴权加密；http.request 模块进行 chunk 的读取和响应。同时使用了表驱动编程 HashMap，映射不同错误码和对应的错误提示，避免多层循环嵌套重复。该项目让我对 Node.js 有了初步的了解和应用。
